# minitms — Svelte + Vite

A tiny local-first, mobile-first PWA for writing and reading short texts (stories, song lyrics).

## Features
- Mobile-first, distraction-free UI
- Local-first: all data persisted to localStorage
- PWA: offline support, installable
- Theme switching: auto, light, dark
- WYSIWYG editing (contenteditable)
- Minimal, book-like reading experience

## Getting Started

```sh
npm install
npm run dev
```

## Project Structure
- `src/` — Svelte components and app logic
- `public/` — static assets, manifest, icons
